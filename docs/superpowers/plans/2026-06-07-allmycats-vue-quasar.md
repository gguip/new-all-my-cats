# AllMyCats — Vue + Quasar Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a single-scroll Vue + Quasar showcase of 18 rescued cats, pixel-faithful to the `design_handoff_allmycats` hifi design, with vue-i18n (pt-BR default + en) and animated personality bars.

**Architecture:** Quasar CLI (Vite) SPA. Quasar provides scaffolding/build/SCSS only — all UI components are custom Vue 3 `<script setup>` SFCs styled with the design's own tokens (no Material components). Design tokens are lifted verbatim from `design_system/tokens/*.css`; the component CSS layer is adapted from `design_system/base.css`. All display copy lives in vue-i18n message files; cat proper names + numeric stats live in a plain data module. Local component state only — no store, no router beyond `/`.

**Tech Stack:** Vue 3, Quasar 2 (CLI Vite), vue-i18n 9, SCSS, Vitest + @vue/test-utils + jsdom.

**Design source (read-only reference):** `/Users/guilhermepassarinho/Downloads/design_handoff_allmycats`

Decoupled from lifecycle for testability; the component calls `start()` on mount and `stop()` on unmount.

```js
import { ref, onMounted, onUnmounted } from 'vue'

export function useNavScroll(threshold = 40) {
  const scrolled = ref(false)
  const onScroll = () => { scrolled.value = window.scrollY > threshold }

  const start = () => {
    onScroll()
    window.addEventListener('scroll', onScroll, { passive: true })
  }
  const stop = () => window.removeEventListener('scroll', onScroll)

  // Auto-wire when used inside a component setup(); no-op in plain tests.
  if (getCurrentInstanceSafe()) {
    onMounted(start)
    onUnmounted(stop)
  }

  return { scrolled, start, stop }
}

// Avoids the Vue warning when onMounted is called outside a component.
function getCurrentInstanceSafe() {
  try {
    // eslint-disable-next-line global-require
    const { getCurrentInstance } = require('vue')
    return getCurrentInstance()
  } catch {
    return null
  }
}
```

```js
import { onMounted, onBeforeUnmount } from 'vue'

// Arms an element for a one-time entrance reveal once it enters the viewport.
// `elRef` is a template ref to the element carrying the `.reveal` class.
export function useScrollReveal(elRef, { failsafeMs = 2600, rootMargin = '0px 0px -8% 0px' } = {}) {
  let observer = null
  let timer = null

  const reveal = (el) => {
    el.classList.add('in')
    cleanup()
  }
  const cleanup = () => {
    if (observer) { observer.disconnect(); observer = null }
    if (timer) { clearTimeout(timer); timer = null }
  }

  onMounted(() => {
    const el = elRef.value
    if (!el) return
    el.classList.add('armed')
    observer = new IntersectionObserver((entries) => {
      if (entries.some((e) => e.isIntersecting)) reveal(el)
    }, { rootMargin })
    observer.observe(el)
    timer = setTimeout(() => reveal(el), failsafeMs) // failsafe: never stay hidden
  })

  onBeforeUnmount(cleanup)
}
```
